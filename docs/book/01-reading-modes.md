# Режимы чтения

В BookStream есть два основных способа читать книгу.

## Book mode

Это режим, в котором чтение ближе к обычной книге:

- текст воспринимается как последовательность страниц;
- пользователь двигается по материалу горизонтально;
- навигация должна ощущаться как листание, а не как прокрутка каталога;
- интерфейс не должен конкурировать с текстом.

Этот режим нужен, когда важны ритм, пауза и чувство отдельной страницы.

## Feed mode

Это режим, который ближе к телеграм-каналу или Telegra.ph:

- чтение идет вертикально;
- текст воспринимается как непрерывная лента;
- навигация становится проще и быстрее;
- режим хорошо подходит для коротких материалов, цитат, пересборок и AI-сводок.

Это не отдельный продукт. Это другая геометрия чтения.

## Почему два режима нужны одновременно

Один и тот же текст может быть прочитан по-разному.

- В book mode пользователь живет внутри книги.
- В feed mode пользователь скользит по смыслу.

Эта разница не косметическая. Она меняет поведение:

- как человек держит внимание;
- как быстро он прыгает между фрагментами;
- насколько легко он цепляет цитату или комментарий;
- где удобнее AI-суммаризация;
- где удобнее показывать короткие версии.

## Принцип интерфейса

Режим чтения не должен ломаться тяжелыми панелями.

У чтения есть право быть спокойным:

- панели появляются только по действию;
- выбор текста вызывает контекстные действия;
- настройки не должны закрывать текст;
- social layer должен быть наложением, а не фоном со статус-баром.

## Что это означает для продукта

BookStream не навязывает единственный способ потребления текста.

Он дает читателю выбрать:

- страницу или ленту;
- длинный текст или короткую версию;
- чистое чтение или чтение с аннотациями;
- спокойный режим или режим обсуждения.
