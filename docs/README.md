# BookStream Docs

Это документация, написанная как книга внутри системы.

Здесь не просто список фич. Здесь описание того, чем BookStream является как продукт: чтение, социальный слой, варианты текста, офлайн, анонимная идентичность, админка и управление книгой как живым объектом.

## Как читать

1. Сначала откройте [пролог](./book/00-prologue.md).
2. Потом идите по главам в любом порядке.
3. Если нужно быстро понять поверхность продукта, начните с [карты чтения](./book/01-reading-modes.md) и [социального слоя](./book/02-social-layer.md).

## Принцип

Документы должны развиваться так же, как и продукт:

- text-first;
- quiet interface;
- one active section at a time;
- no noisy catalogue style;
- social features as a layer over reading, not a replacement for reading.
